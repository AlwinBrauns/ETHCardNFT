# Work in Progress

## How To:
1. Install MetaMask
2. ```npm install```
3. ```npx hardhat compile```
4. ```npx hardhat node```
5. Import Account into MetaMask (private key)
6. ```npx hardhat run scripts/deploy-cards-contract.js --network localhost```
7. Import address (src/address.json) into MetaMask to see NFT
8. ```npm run dev```
